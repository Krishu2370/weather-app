# Weather Forecast App

A simple weather forecast app that shows real-time weather updates using a public API.

##  Features
- Search weather by city
- Displays temperature, humidity, weather condition
- Responsive and clean UI

## Technologies Used
- HTML, CSS, JavaScript
- OpenWeatherMap API
